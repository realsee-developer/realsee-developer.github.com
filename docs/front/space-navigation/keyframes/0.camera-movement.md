---
title: 相机运镜
---

**相机运镜** 控制相机实现镜头位移、旋转、模态切换等模拟如视 3D 空间漫游的能力。

## 字段说明
```ts
type CameraMovementData = {
  effect: CameraMovementEffect // 运镜效果
  mode: Mode // Five.Mode
  panoIndex: number // 目标点位
  loop?: boolean // 是否循环
  rotateSpeed?: number // 旋转速度
  rotation?: Rotation // 旋转方向
} & Partial<Pose> // Five中定义的相机视角信息
```
