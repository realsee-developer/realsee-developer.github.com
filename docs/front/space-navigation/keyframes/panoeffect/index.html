全景特效

功能说明​

全景特效 ( PanoEffect ) 指在如视 3D 空间全景游走模态下执行某些特效。

info

目前仅支持测量长度的特效。比如，在全景游走模态下标注出窗户的宽度信息。

效果展示​

字段说明​

  • effect: 全景特效类型，其类型为枚举类型 PanoEffectEnum 。
    • PanoEffectEnum.Distance: 两点间的距离。
  • twoVertexs: 空间中两个点的三维坐标数组。
info

具体内容参数类型可以参见 Vreo API 文档。

全景特效类型数据样例
{
  "uuid": "cbd4326c-3cb1-411e-348a-55a6fa42d6d0",
  "type": "PanoEffect",
  "start": 3578,
  "end": 25939,
  "parsed": false,
  "data": {
    "twoVertexs": [
      {
        "x": -3.216338116298317,
        "y": 1.221307932920801,
        "z": -3.991326093673706
      },
      {
        "x": -3.178751302976147,
        "y": 0.09148991767376025,
        "z": -3.991326093673706
      }
    ],
    "effect": "Distance"
  }
}
