title: 空间文本
---

## 功能说明
**空间文本** ( PanoTextLabel ) 在如视 3D 模型空间中某一个三维坐标点上展示的一段文本。

## 效果展示
## 字段说明
- `text`: 空间内需要展示的文本内容。
- `vertex`: 空间内展示文本的三维坐标点。
- `fontSize`: 可选字段，设置文本字体大小，默认为 16px 。

```json title="空间文本类型数据样例"
{
  "uuid": "eb7b3d3e-5f13-423d-8270-c4bfa4201d0b",
  "type": "PanoTextLabel",
  "start": 71200,
  "end": 73200,
  "parsed": false,
  "data": {
    "text": "档位随烟量自动调节",
    "vertex": {
      "x": -2.1652547611365116,
      "y": 1.612348936240625,
      "z": -8.787925732476422
    },
    "fontSize": 14
  }
}
```
