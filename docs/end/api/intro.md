如视（[realsee.com](https://realsee.com)）依托成熟的三维重建/深度预测/WebGL渲染等技术，利用标准、易用的 SDK 及配套 OpenAPI，来实现空间的线上化以及沉浸式的用户体验。

由于用户终端的实现基于浏览器/WebView组件，因此跨平台特性得天独厚。现如今，我们将这些能力以开放平台的形式对外输出，便于行业从业者能够快速搭建一整套生动、有趣的 3D/VR 应用。
