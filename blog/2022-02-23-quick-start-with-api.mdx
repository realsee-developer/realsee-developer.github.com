## 个人尝鲜

[**如视VR**](https://apps.apple.com/cn/app/id1502262095) 是如视官方提供的一款全景模型拍摄神器——您可以通过手机按照指导进行简单拍摄，拍摄处理后会生成 VR 实景数据。

生成的 VR 数据您可以通过如视开放平台的 [**开放 API**](/open/api/#/) 来获取，然后利用 [**渲染引擎 Five**](/docs/front/3d-space/get-started/rendering-engine) 就可以打造您自己的三维空间应用。

那么跟随本文，快速“尝鲜”如视的三维空间能力吧。

## 30分钟跑通采集预览

:::tip 提示
我们假设你对 [JavaScript](https://developer.mozilla.org/en-US/docs/Web/JavaScript/A_re-introduction_to_JavaScript) 语言、[Node.js](https://nodejs.org/en/docs/) 及 [npm](https://docs.npmjs.com/) 等知识已有基本的了解。
如果您之前没接触过，建议阅读本文前简单了解下。
:::

### 准备工作

- 您的机器已安装 [Node](https://nodejs.org/en/) 环境。
- 您方便操作的苹果手机——用于拍摄，采集 VR 数据。

### 拍摄采集

首先，你需要从苹果应用商店下载 [**如视VR**](https://apps.apple.com/cn/app/id1502262095) ——在应用商店搜索 **"如视VR"**。按照指引注册帐号、登录后，继续在 App 内操作：

- 执行 **"我的"** ➜ **"身份选择"** ➜ **"个人"**（默认）。
- 点击底部栏 **"+"** 加号按钮 ➜ 选择 **"手机拍"**（暂时仅支持iOS）。
- 填写项目名称后，点击 **"创建项目"** 即可进入采集流程。

按照 App 操作指引完成拍摄并上传，等待 VR 项目处理完成（此过程大致需要几分钟）。

:::tip 提示
建议您选择光线好的室内场景，比如选择在白天拍摄自己的小卧室。
:::

### 获取数据

完成拍摄并上传之后，你可以通过如视 [开放 API](/open/api) 获取数据。在请求数据之前，您需要登录[如视开放平台控制台](https://developers.realsee.com/console)——帐号就是您刚刚登录 App 注册的。

然后点击 **"申请密钥"**，获取您的 `Application Id` 和 `Application Secret`，此凭证将用于访问 API 接口获取空间数据。

涉及的 API 有：

- 鉴权：`/auth/access_token`。
- 列表数据：`/open/v1/entity/vr/list`。
- 基础数据：`/open/v1/entity/vr`。

:::tip 提示
发起任意 API 调用时，先获取临时 `access_token`（默认五分钟），后续请求均携带。业务方可以考虑设置约三分钟的缓存，以减少整体服务端 API 的响应时长。
具体发起流程可参考源码 [five-sdk-starter-with-api](https://github.com/realsee-developer/five-sdk-starter-with-api) 。
:::

### 渲染 VR

为节约您创建开发脚手架的成本，你也可以 `git clone` [five-sdk-starter-with-api](https://github.com/realsee-developer/five-sdk-starter-with-api) 。

进入仓库目录打开 `index.js`，搜索`Your-App-Key` 字符串替换成申请的`Application Id`，搜索 `Your-App-Secret` 字符串替换成申请的 `Application Secret`。

最后，开启您的终端：

```shell
cd five-sdk-starter-with-api
npm i
npm run start
# 您会看到控制台输出，浏览器打开下面的链接，建议使用Chrome浏览器保证最佳体验
> node index.js
> serve at http://127.0.0.1:3000
```

至此，访问 [127.0.0.1:3000](http://127.0.0.1:3000) 就可以体验您自己拍摄的 VR 空间啦~

:::tip 提示
如果您想更深度地去定制自己的 VR 空间应用，访问 [**渲染引擎 Five**](/docs/front/3d-space/get-started/rendering-engine) 了解更多。
:::
